# URL Shortener GUI

This Python script provides a simple Graphical User Interface (GUI) for shortening URLs using the TinyURL service. It uses the `pyshorteners` library for URL shortening and `pyperclip` for copying the shortened URL to the clipboard.

## Features

- Enter a long URL in the input field.
- Click the "Generate Short URL" button to shorten the URL using TinyURL.
- The shortened URL will be displayed in the output field.
- Click the "Copy URL" button to copy the shortened URL to your clipboard.

## Prerequisites

- Python 3.11 or later installed on your system.
- Required Python libraries (`pyperclip`, `pyshorteners`, `tkinter`) are installed.
